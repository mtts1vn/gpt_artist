## How to use:

To generate a pixel art in this app using the Chat GPT, use this prompt to get the pixel art code:

Create a pixel art using the following colors:
0: Colors.white,
1: Colors.red,
2: Colors.green,
3: Colors.blue,
4: Colors.pink,
5: Colors.brown,
6: Colors.purple.

Please provide the dimensions of the pixel art and a description of what you would like to be drawn.

After receiving the description, please send only the sequence of numbers corresponding to the colors for the pixel art, respecting the order of pixel positions.

Please give me only the sequence of numbers and separate each number with a comma.
